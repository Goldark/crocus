####################
Crocus Documentation
####################

Crocus is a small and experimental async web microframework built on top of aiohttp (required python 3.5+).

  .. code:: bash

    pip install crocus


- :ref:`introduction_ref_label`: framework introduction, installation, features, etc;
- :ref:`api_reference_ref_label`: full framework API reference;
- :ref:`examples_ref_label`: code samples;
- :ref:`changelog_ref_label`: framework changelog list.
